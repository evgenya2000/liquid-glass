# Liquid Glass — эффект «жидкого стекла» на чистом CSS

Демонстрация визуального стиля Liquid Glass, вдохновлённого интерфейсами Apple.  
Эффект создаётся с помощью `backdrop-filter` и SVG-фильтра с картой смещения, без JavaScript и сторонних библиотек.

## Как это выглядит

- 🧊 **Обычное стекло** — равномерное матовое размытие (`blur`)
- 🌊 **Liquid Glass** — искажённый фон с микроволнами, словно тонкая живая плёнка

## Структура проекта

├── index.html # разметка, SVG-фильтр, две демокарточки
├── styles.css # стили, фон, эффекты стекла
├── cat.png # (опционально) фоновое изображение
└── README.md

## Поддержка браузерами

✅ Chrome / Edge — полная поддержка SVG-фильтра в backdrop-filter

⚠️ Safari — displacement-карты не поддерживаются, эффект деградирует до обычного размытого стекла

⚠️ Firefox — может потребоваться флаг layout.css.backdrop-filter.enabled

## Как это работает
SVG-фильтр #displacementFilter использует встроенную PNG-карту шума (base64) для смещения пикселей фона. Красный и зелёный каналы задают векторы искажения по осям X и Y, а небольшой blur() смягчает резкость.

## Настройка
Интенсивность искажений — измените атрибут scale у <feDisplacementMap> в index.html

Цвет и прозрачность стекла — поправьте background: rgba(...) у .glass и .liquid-glass в styles.css

Фон — замените cat.png на своё изображение или верните градиент
